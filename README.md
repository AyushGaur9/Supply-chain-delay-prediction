# End-to-End Supply Chain Analysis

An end-to-end supply chain analytics project focused on delivery performance, operational bottlenecks, and shipment delay prediction using Python-based data analysis and machine learning techniques.

## Project Overview

This project analyzes a large-scale Kaggle supply chain dataset containing 180k+ records to uncover patterns affecting shipment delays and order profitability.

The analysis includes:
- Delivery performance evaluation across regions, shipping modes, and time segments
- Profitability impact analysis of delayed shipments
- Root cause analysis for operational bottlenecks
- Predictive modeling to identify high-risk orders before shipment

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Key Features

### Exploratory Data Analysis (EDA)
- Analyzed delivery trends and shipping performance
- Visualized delay patterns across categories and regions
- Measured profitability impacts caused by late deliveries

### Root Cause Analysis
- Identified operational factors contributing to shipment delays
- Evaluated shipping modes and regional performance differences

### Predictive Modeling
- Built a machine learning model to predict high-risk delayed orders
- Generated actionable insights to improve on-time delivery rates

## Dataset

- Kaggle Supply Chain Dataset
- 180,000+ records analyzed
